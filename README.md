# 📊 HR MIS Dashboard (Excel-Based)

## 🔍 Objective

The objective of this project is to analyze employee data and monitor attrition trends, workforce distribution, and performance metrics using Excel-based dashboards.

---

## 📁 Dataset Overview

The dataset contains employee-level information across departments, locations, and roles.

### Key Attributes:

* Employee ID & Name
* Department (HR, IT, Sales, Finance, Operations)
* Location (Bangalore, Mumbai, Nagpur, Pune)
* Joining Date & Exit Date
* Employee Status (Active / Exited)
* Salary
* Performance Rating
* Attendance (%)
* Attrition Indicator

---

## 🛠 Tools Used

* Microsoft Excel

  * Pivot Tables
  * Pivot Charts
  * Slicers (Month, Department, Location)
  * Conditional Formatting
  * KPI Cards using formulas

---

## 📊 Key KPIs

* **Total Employees:** 500
* **Active Employees:** 247
* **Attrition Rate:** 50.60%
* **Average Salary:** ₹7.12 LPA
* **Average Attendance:** 87.30%
* **Highest Attrition Department:** Finance

---

## 📈 Dashboard Features

### 🔹 Attrition Analysis

* Monthly attrition trend
* Attrition comparison across departments
* Peak attrition observed in June

### 🔹 Workforce Insights

* Headcount distribution by department
* Joined vs Resigned comparison
* Active vs Exited employees tracking

### 🔹 Performance Analysis

* Performance distribution (High vs Low)
* Impact of performance on attrition

### 🔹 Department-Level Metrics

* Attrition % by department
* Attendance analysis by department

---

## 📊 Key Insights

* Overall attrition rate is **high (50.6%)**, indicating retention issues
* **Finance department** shows the highest attrition (~55.56%)
* Attrition peaks in **June**, suggesting possible seasonal trends
* **Low-performing employees contribute majority of exits**
* Hiring is not keeping pace with attrition
* Attrition is consistently high across departments
